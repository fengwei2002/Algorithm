### 思路

注意 `continue` 的使用，还有典中典的去重 while

### 代码

``` cpp
// Leetcode 题解区搜索 fengwei2002
// https://github.com/fengwei2002/algorithm

class Solution {
  public:
    int countSegments(string s) {
        int ans = 0;
        for (int i = 0; i < s.size(); i ++ ) {
            if (s[i] == ' ') continue;
            int j = i + 1;
            while (j < s.size() && s[j] != ' ') j++;
            ans++;
            i = j - 1;
        }
        return ans;
    }
};
```
