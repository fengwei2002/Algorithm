### 解题思路

把 int 分为 8 段

每段 4 位， 把每段都转换为对应的数字

然后将结果逆序

### 代码

```cpp
class Solution {
public:
    string toHex(unsigned int num) {
        if (!num) return "0";
        string ans, nums = "0123456789abcdef";

        while (num) {
            ans += nums[num & 0xf];
            num >>= 4;
        }

        reverse(ans.begin(), ans.end());
        return ans;
    }
};
```
