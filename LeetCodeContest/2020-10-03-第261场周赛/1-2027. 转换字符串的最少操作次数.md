### 解题思路

贪心算法：从第一个是X开始消除，是最优。从第二个、第三个消除都不是最优。


### 代码

```cpp
class Solution {
public:
    int minimumMoves(string s) {
        int ans = 0;
        for (int i = 0; i < s.size(); i++) {
            if (s[i] == 'X') {
                ans++;
                i += 2;
            }
        }
        return ans;
    }
};

```
