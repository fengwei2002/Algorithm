/* 给定一个正整数，请你求出 1-n 中质数的个数 */

朴素筛法： 从前到后，将所有数字的倍数全部删除

``` cpp
#include <algorithm>
#include <iostream>

using namespace std;

const int N = 1000010;

int primes[N], cnt;
bool st[N];

void get_primes(int n) {
    for (int i = 2; i <= n; i++) {
        if (st[i] == false) {
            primes[cnt++] = i; // 如果不是质数的话， 加到数表里面
            for (int j = i + i; j <= n; j += i)  {
                st[j] = true;
            }
        }
    }
}

int main() {
    int n;
    cin >> n;
    get_primes(n);
    cout << cnt << endl;
    return 0;
}
// o(log_n)
```
 
当一个数字不是质数的时候我们就不需要删除它的所有倍数   
1-n 中有 $\frac{n}{ln_n}$ 个质数

线性筛法：每一个 n 只会被最小质因子筛掉 ，时间加快一倍左右

``` cpp
#include <iostream>
#include <algorithm>

using namespace std;

const int N= 1000010;

int primes[N], cnt;
bool st[N];

void get_primes(int n) {
    for (int i = 2; i <= n; i++) {
        if (st[i] == false) {
            primes[cnt++] = i; // 将这个数加到 数表中
        }
        // 对数表中的元素进行遍历
        for (int j = 0; primes[j] <= n / i; j++) {
            st[primes[j] * i] = true; // 将数表中 元素 * i 的数字删除 

            if (i % primes[j] == 0) break; 
            //这句话成立的时候， primes[j]一定是 i 的最小质因子
            // 所以 primes[j] 一定是  primes[j] * i 的最小质因子，
        }
    }
}

int main() {
    int n;
    cin >> n;
    get_primes(n);
    cout << cnt << endl;
    return 0;
}
```
