## 思路

`f[i][j]` 表示现在有 j 个鸡蛋，i 层楼最坏情况所需要最少次数

`f[i][j] = f[i - 1][j] + f[i - 1][j - 1] + 1`

## 代码


``` cpp 
#include <iostream>
using namespace std;

int f[101][11], n, m;
// n 层楼， m 个鸡蛋，最坏情况扔鸡蛋次数最少的策略

int main() {
    f[0][0] = 0;
    while (cin >> n >> m) {
        for (int i = 1; i <= n; i++) {
            for (int j = 1; j <= m; j++) {
                f[i][j] = f[i - 1][j] + f[i - 1][j - 1] + 1;
            }
            if (f[i][m] >= n) {
                cout << i << endl;
                break;
            }
        }
    }
    return 0;
}
```
