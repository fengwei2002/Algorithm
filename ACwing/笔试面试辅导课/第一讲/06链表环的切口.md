## 代码

首先判重的第一反应就是 哈希表，开一个哈希表，如果这个节点之前已经存在，那么就直接返回


``` cpp 
class Solution {
   public:
    ListNode *entryNodeOfLoop(ListNode *head) {
        if (head == nullptr) return head;

        unordered_map<ListNode*, int> hash;
        auto p = head;
        
        while (p) {
            if (hash[p]) {
                return p;
            }
            hash[p] = 1;
            p = p->next;
        }
        
        return nullptr;
    }
};
```
